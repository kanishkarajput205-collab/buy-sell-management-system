# 🛒 Buy & Sell Management System

A desktop-based Buy & Sell Management System developed using **Java Swing** and **Object-Oriented Programming (OOP)** principles.

## 📌 Overview

This application allows users to manage buying and selling operations through a graphical user interface. It demonstrates core Java programming concepts including object-oriented programming, event-driven programming, multithreading, and GUI development.

---

## ✨ Features

- User Login
- Buyer Management
- Seller Management
- Buy Products
- Sell Products
- Inventory Management
- Search Functionality
- Graphical User Interface (Java Swing)

---

## 🛠 Technologies Used

- Java
- Java Swing
- Object-Oriented Programming
- Multithreading
- File Handling
